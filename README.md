## Setup

### Testing DCL

- Run these commands from within the `scene` directory
- Run `npm install`
- Run `dcl start`

### Testing the image processor

- Run these commands from within the `utils` directory
- Run `npm install`
- Run `npm run start` to process the image and return a json blob
